### Run Locally

To run **Transportio** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/transportio.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/transportio.git
```
